Statistics

A summary statistics of the given data.

Average Distance Traveled

I calculated the average distance traveled by a rider to be 3.86 miles. To calculate this I used a library in R called geosphere which contains a function called 'distm' which computes the distance between two coordinates. In order to calculate the average, I had to remove all data which contained any null entries, as the calculating the distance was not plausible in those cases.
