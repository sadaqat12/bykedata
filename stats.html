Statistics

A summary statistics of the given data.

Average Distance Traveled

The average distance traveled by a rider was calculated to be 3.86 miles. To calculate this a library in R called geosphere which contains a function called 'distm' was used, which computes the distance between two coordinates. In order to calculate the average, I had to remove all data which contained any null entries, as the calculating the distance was not plausible in those cases.

This segmented bar graph shows the breakdown of the Trip Route - Passholder Type combinations. It can be seen that for all the categories of the passholder one way going clients outnumber the returning clients. I believe a cause of this could be that people in the monthly pass holding category for example could be using the bikes as a means to get to work, and back which are one way trips. While there are more clients who return bikes to their starting points in the walk-up category, which could be because walk-up clients have trips that are more spontaneous such as going to the corner store.
